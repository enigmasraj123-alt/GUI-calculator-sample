🧠 Math Solver AI

A simple and powerful web-based math solver built using Python, SymPy, and Gradio.# GUI-calculator-sample

🚀 Features
✅ Solve algebraic equations
📉 Compute derivatives
∫ Perform symbolic integration
🔢 Numerical integration with bounds
🌐 Clean web interface using Gradio
🛠️ Tech Stack
Python
SymPy
NumPy
SciPy
Gradio
📂 Project Structure
math-ai/
│
├── main.py          # Entry point
├── engine.py        # Math logic (SymPy)
├── gui.py           # Web interface (Gradio)
└── requirements.txt # Dependencies
⚙️ Installation
Clone the repository:
git clone https://github.com/en1gma_alt/math-ai.git
cd math-ai
Install dependencies:
pip install -r requirements.txt
▶️ Run the App
python main.py

Then open the browser link shown in terminal (usually http://127.0.0.1:7860)

🧪 Example Inputs
x + 5 = 20
x**2
sin(x)*exp(x)
x**3 + 2*x
